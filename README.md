# Installation
To install either
1. Download the EXE
2. Build from source

To do option 2:
1. run ``git clone https://github.com/dumpydev/ZoomFcker``
2. cd into the directory
3. run ``npm i`` to install dependencies
4. run ``npm i -g pkg`` to install pkg
5. run ``pkg .``
6. Run the EXE.
